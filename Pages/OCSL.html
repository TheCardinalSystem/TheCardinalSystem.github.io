Usage:


To create a store you can use the following code:
public static final Config config = new Config("config_name");

Adding values to the store can be done as follows:

Single Object:

2
3
//import com.Cardinal.Obj.ConfigElement;
ConfigElement element = new ConfigElement("name", "value"); 
config.addElement(element);

Array Object:

//import com.Cardinal.Obj.ArrayElement; 
String[] array = {"val1", "val2", "val3"}; 
ArrayElement subElement = new ArrayElement("name", array);
ConfigElement element = new ConfigElement(subElement);
config.addElement(element);

//Returns new ConfigElement 
ConfigElement element = config.getElement("name"); 
//Submethods of ConfigElement are as follows:
element.setName("name");
//Changes the element's name 
element.setValue("value");
//Changes the element's value 
element.getName();
//Returns a String with the element's name
 element.getValue();
//Returns a String with the element's value
 element.getElementType();
//Returns 0 for regular objects and 1 for arrays 
element.getArrayElement();
//Returns the ArrayElement used in the constructor. Only used if getElementType returns 1 
//Submethods of ArrayElement are as follows:
element.setName("name");
//Changes the element's name
element.setValues(array);
//Changes the element's values 
element.getName();	
//Returns a String with the element's name element.getValue();
//Returns a String[] with the element's values
